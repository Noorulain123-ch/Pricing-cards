#  Pricing Cards UI

A clean and responsive **Pricing Cards UI** built using HTML, CSS, and JavaScript. This project showcases different subscription plans in an attractive and user-friendly layout.

##  Features

*  Multiple pricing plans (Basic, Pro, Enterprise)
*  Highlighted "Most Popular" plan
*  Feature comparison with enabled/disabled states
*  Modern and responsive design
*  Testimonial section for user feedback
*  Call-to-action buttons for each plan

## Technologies Used

* HTML
* CSS
* Font Awesome (for icons)


##  How It Works

* Users can view different pricing plans
* Compare features across plans
* Identify the "Most Popular" plan easily
* Click **Get Started** buttons for actions (can be extended with functionality)
